Life Cycle Methods:

Life cycle of a Component has three phases:

Mounting Phase

Updating Phase

Unmounting Phase

Each phase will have methods which are invoked in a sequential manner

Mounting phase:

1. Constructor:

Constructor is a special type of function which is invoked at the moment when component is created.

Constructor is used to initialise the state variables

2.Static getderived state from props:

This method is invoked to return a object for updating the state or return null to update nothing.

3.Render:

This method is invoked to return the JSX

4.Componentdidmount:

Componentdidmount is invoked when the component is successfully inserted in the dom.


Updating phase:

1.Static getderived state from props:

This method is invoked to return a object for updating the state or return null to update nothing.

2.shouldComponetupdate:

This method is invoked before Render

3.Render:

This method is invoked to return the JSX

4.getSnapShotbeforeupdate:

This method is invoked to capture the component's information before it gets updated

5.Componentdidupdate

Componentdidupdate is invoked when the component is successfully updated


Unmounting Phase:

Componentwillunmount:

Componentwillunmount is invoked when the componet is removed from the dom.
